# Labyrinth-Prototype-Release
A puzzle game in work, featuring fully one-way tiles and step based logic. The source code is private, only executables available here.

## Log location

for Windows, the log is located in `%APPDATA%\Godot\app_userdata\labyrinth-prototype`, which is usually actually `C:\Users\[your username]\Appdata\Roaming\Godot\app_userdata\labyrinth-prototype\logs`.
